# AI-Driven Human Pose Tracking & Visualization

## Project Description
This project combines **Artificial Intelligence and Computer Vision** to visualize dance movements using **MediaPipe Pose Detection** and **OpenCV**. It tracks human body poses from a dance video and generates a glowing, mirrored skeleton animation with dynamic color effects, transforming dance motion into creative visual art.

## Project Vision
The vision of this project is to demonstrate how AI and Computer Vision can be used beyond traditional analytics for **creative expression**. By blending pose estimation with visual effects, the project highlights how technology can convert human motion into interactive and artistic digital experiences.

## How It Works
The input dance video is processed frame-by-frame using OpenCV. MediaPipe Pose Detection extracts key body landmarks, which are then connected to form a skeletal structure. The skeleton is mirrored, enhanced with glow effects, and rendered with dynamic color changes to create visually engaging animations.
