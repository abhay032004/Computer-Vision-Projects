# Computer Vision Projects Using OpenCV & AI

## Repository Overview
This repository contains a collection of **Computer Vision projects** developed using Python, OpenCV, and AI-based techniques. The projects focus on extracting meaningful information from images and videos, enabling machines to understand and interpret visual data.

Each project demonstrates practical applications of computer vision in real-world scenarios such as pose detection, object detection, and visual analysis.

---

## Key Skills Demonstrated
- Image and video processing
- Pose detection and tracking
- Object detection and visualization
- Feature extraction and visual analytics
- AI-driven computer vision workflows

---

## Tools & Technologies
- Python
- OpenCV
- MediaPipe
- NumPy
- YOLO (Object Detection)
